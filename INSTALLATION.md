# Installation Instructions

Follow these steps to install the project:

1. **Clone the repository**
   ```bash
   git clone https://github.com/enockIT-dev/enock.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd enock
   ```

3. **Install the necessary dependencies**
   - If using npm:
     ```bash
     npm install
     ```
   - If using yarn:
     ```bash
     yarn install
     ```

4. **Configure environment variables**
   - Create a `.env` file and fill it with the required configuration. Refer to `.env.example` for available options.

5. **Run the application**
   ```bash
   npm start
   ```

For further assistance, please check the documentation or contact the support team.
